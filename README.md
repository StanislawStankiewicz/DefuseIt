## I2C Codes

## Startup sequence

- BFS looking for modules
- each module responds with own unique address
- master saves addresses

## Technical requirements

Each module needs 4/5 connections on each side:

- 5/3.3 V
- GND
- SDA
- SCL
- INT (optional)
